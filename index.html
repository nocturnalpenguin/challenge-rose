Rose (Rhodonea Curve)

A curve which has the shape of a petalled flower. This curve was named rhodonea by the Italian mathematician Guido Grandi between 1723 and 1728 because it resembles a rose.

this curve is a sinusoid plotted in polar coordinates and uses the following equation:

r = cos(kθ)

The shape of the graph is strongly dependent on the value of k.

k = n / d
